Tags título: h1 - h6

Voltar

As tags h1 à h6 definem títulos hierarquicamente. A tag h1 define o título de maior importância na página, enquanto o h6 define o título de menor relevância. Não é recomentado pular nenhum título, mas, utiliza-los em ordem.

Tags para inserção de texto: p, blockquote, q, small, article, aside, abbr

Voltar
“Um dos meus dias mais produtivos foi quando eu joguei fora 1000 linhas de código.”

O HTML tem tags próprias para inserção de cada tipo de texto. Verifiquemos algumas destas tags a seguir:
A tag 'p' serve para inserir um parágrafo na página.
A tag 'small' serve para inserir um texto bem pequeno na página, como símbolo de copyright ou o botão volta, presente neste texto.
A tag 'aside' serve para adicionar informações que não são fundamentais ao texto. Ao se falar sobre vendas de carros, pode-se adicionar um aside sobre pneus, por exemplo.
A tag 'article' é um texto completo e independente. São matérias de jornais, por exemplo, que não dependem de outras partes do site para serem compreendidas.
A tag 'blockquote' serve para inserir uma citação longa na página. Por sua vez, a tag 'q' serve para inserir uma citação bem curta.
A tag 'abbr' serve para inserir uma abreviação ou um acrônimo no arquivo HTML. Nesta frase, a palavra HTML foi escrita com esta tag.

Tags para formatação de texto:i, u, b, strong,font, sub e sup

Voltar

Ao longo do curso, aprendemos sete tags de formatação de texto, de maneira mais focada. São elas:
A tag 'i' torna o texto itálico e a tag 'u' coloca um sublinhado no texto. As tags 'b' e 'strong' são similares mas possuem diferença semântica. Enquanto a tag 'b'. apenas coloca o texto em negrito. A tag 'strong' coloca o texto em negrito e enfatiza a relevância daquele determinado trecho do texto.
A tag 'sup' define um texto acima do texto regular, como a pontecia da matemática. Enquanto a tag 'sub' define um texto abaixo do texto regular, como o número 2 em h2o. A tag 'font' servia para definir a fonte do texto. Contudo, esta tag caiu em desuso após o HTML5.

Tag para exclusão do texto: del e s

Voltar

Existem dois tipos de tags para exclusão no HTML: del e s.
A tag 'del'sinaliza uma parte deletada do texto. É comum que ela seja usada com a tag 'ins' que insere um substituto para tal trecho deletado. Essa tag tem dois atributos: 'cite', no qual é possível citar a url que explica a razão da exclusão e 'datetime', que especifíca a hora e o dia da exclusão.
A tag 's' marca um texto que já não está correta, mas não o exclui. Ao invés disso, o texto aparecerá taxado.

Tags de mídia: audio, video, figure, iframe, img

Voltar

A tag 'audio' é usada para anexar áudios à pagina HTML. Esta tag aceita três formatos: mp3, WAV e OGG. É recomendado adicionar mais de um source de áudio para caso o primeiro não funcione. Alguns navegadores podem não aceitar a tag audio. Neste caso, é interessante incluir uma mensagem na tag, que será exibida em caso de defeito. A tag 'audio' contém diversos atributos: autoplay, controls, loop, muted, preload e o src.
A tag 'video' é usada para anexar vídeos à página e tem regras de funcionamento similares ao àudio. Esta tag suporta videos MP3, WebM e OGG. Para além dos atributos autoplay, controls, loop, muted e src. A tag 'video' também conta com os atributos height e width, que permitem redimensionamento, preload e poster, que permite add um imagem a ser mostrada enquanto o vídeo carrega.
A tag 'img' é usada para anexar imagens à pagina HTML. Esta tag requere os atributos src, onde irá o endereço da imagem, e alt, descrição da imagem que aparece quando a imagem não carrega. Esta tag tem diversos atributos: alt, src, crossorigin, height, width, ismap, loading, determina quando a imagem deve ser carregada, longdesc, determina uma url para uma dewcrição mais detalhada, referrerpolicy, size, srcset e usemap
A tag 'iframe' é uma tag usada para indexar documentos externos ao html, como por exemplo, sites. É importante sempre add um atributo 'tittle' para ajudar leitores de tela. O iframe tem diversos atributos:allow, allowfullscreen, allowpaymentrequest, height,width,loading, name, referrerpolicy, sandbox, src, srcdoc. A tag 'figure' é uma tag que "abraça" conteúdos relacionados à imagem e a sua legenda, como diagrama, imagem, video etc.

Tags de divisão:br, div, span e section

Voltar

A tag 'br' insere uma quebra de linha. Este recurso tem sido usado em todos os espaçamentos de definição de tags neste arquivo.
A tag 'div' é uma divisão ou uma seção do HTML. Ela não é semanticamente recomendada por não carregar muito significado mas é útil para estilização e containerização.
A tag 'span' é um container em linha para marcar uma parte do documento. Enquanto a div marca blocos da página, a span permite marcar unidades menores como uma letra, um setor etc.
A tag 'section' cria uma seção no documento.

Tags de conexão: a, button e base

Voltar

A tag 'a' define um hyperlink.
A tag 'base' define uma url base ou uma url alvo para a qual apontará todas as urls do documento.
A tag 'button' define um botão clicável. Há diferentes tipos de botões: reset, submit, button.

Tags de legendagem: caption, figcaption e track

Voltar

A tag 'caption' define a legenda de uma tabela. Ela deve ser inserida imediatamente após a tag 'table' e normalmente aparece centralizada.
A tag 'figcaption' define a legenda de uma foto. É usada junto da tag 'figure'
A tag 'track' é usada para legendar vídeos. Para ela funcionar, é necessário colocar um arquivo vtt. com o texto e a minutagem da legenda. Ela tem alguns atributos: default, kind, label, src (obrigatório), srclang.

Tags de tabela e lista: table, th, tbody, tfooter, tr, td, li, ul e ol

Voltar

As tabelas em HTML são construídas dentro da tag 'table'. Contudo, esta tag não implementa uma tabela se usada isoladamente. É necessária a implementação de linhas (rows) e colunas (collumns). Para tanto usamos as tag tr (row) e td (collumns). Estas tags apresentam os dados da tabela. Para a tabela ficar mais semântica, também colocamos o título de cada coluna através da tag 'th' (head).
A tabela também pode ter suas seções dividas. Para tanto, utilizamos as tags 'thead' para definir os elementos de título, 'tbody' para implementar a parte da dados (é obrigratório conter pelo menos um tr dentro) e 'tfooter', onde ficará elementos como o total de uma tabela de cálculo.
Além da tabela, existe outra formas de apresentar dados em HTML: as listas. Existem dois tipos de lista. As listas ordenadas vem acompnhadas de números e são introduzidas pela tag 'ol'. POr sua vez, as listas não ordenadas tem apenas marcadores e são introduzidas pela tag 'ul'. Uma lista precisa conter itens. A tag 'li' é responsável por introduzir dados à lista.
